
The built-in Windows "automatically hide task bar" setting is not useful with a touchscreen, because you can easily swipe up from the bottom to bring it back. 

Use the freeware [thide.exe](https://github.com/amnweb/thide) to visibly remove the taskbar (as long as this application is running).

Then, open Windows Settings and search for "gesture." Disable all of the gestures, especially "two-finger gestures" and "swipe from edge gestures."
